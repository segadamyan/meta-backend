## Steps to run the app

### 1. Install `Requirements`

```bash
pip install -r requirements.txt
```

### 2. Make migrations

```bash
python3 manage.py makemigrations
```

### 3. Migrate

```bash
python3 manage.py migrate
```

### 4. Run the app

```bash
python3 manage.py runserver
```
